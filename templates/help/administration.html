    Администрирование в {% get_project_name %}
    {% endblock card-title %} {% block card-content %}

    Для администраторов доступна вкладка Администрирование в {% get_project_name %}, где можно управлять пользователями и группами.

    Существует 3 подраздела в ней:
    1) Сведения. Здесь можно посмотреть общую информацию. Например, последняя активность на сайте.
    2) Список групп. Представляет из себя список всех групп в системе. Группы можно создавать, редактировать, удалять и просматривать.
    3) Список пользователей. Коллекция всех пользователей в приложении. Здесть пользователей можно создавать, редактировать, удалять.

    Создание пользователя


    В вкладке Список пользователей по нажатию кнопки Добавить пользователя можно перейти к созданию новой учетной записи.

    Для создания нового пользователя необходимо заполнить поля:
    1) Имя пользователя.
    2) Фамилия пользователя.
    3) Имя пользователя для входа (Username).
    4) Адресс электронной почты.
    5) Пароль.
    6) Подтверждение пароля.
    7) Роль. Определяет права в системе.

    Необязательными полями являются:
    1) Фотография пользователя. Квадратного размера.
    2) Группы пользователя. Например, администратору необязательно иметь группу и он все равно может пользоваться всеми воможностями.

    После нажатия кнопки Добавить пользователь будет создан в системе и его можно будет найти в Списке пользователей.

    Редактирование / удаление пользователя


    В Списке пользователей по нажатию на пользователя будет раскрыто меню, где можно отредактировать учетную запись или удалить.

    Интерфейс редактирования пользователя эквивалентен Созданию пользователя.

    Группы


    В Списке групп отображаются все группы системы. По нажатию на Добавить группу можно создать новую группу.

    Группы необходимы для логической группировки пользователей, чтобы затем можно было открывать тесты для пользователей определенных групп.

    Пользователь может находится в нескольких группах одновременно.

    • Администрирование
    • Создание пользователя
    • Редактирование / удаление
      пользователя
    • Группы
